# Python ETL Pipeline

## Overview
This repository contains a Python ETL pipeline that loads CSV extract files into a MySQL database. It also includes SQL queries used to analyze the imported data and serves as the data source for a Power BI dashboard.

## Project Structure

```
Create_SQL_Tables/
    SQL scripts for creating database tables

Extracts/
    Sample CSV files used in the ETL process

Python_ETL/
    Python scripts for loading and transforming data

SQL_DQL_Queries/
    SQL queries used for data analysis

README.md
```

## Technologies Used

- Python
- MySQL
- SQL
- Power BI
- Pandas
- MySQL Connector

## ETL Process

1. Read CSV extract files.
2. Create MySQL database tables.
3. Clean and transform the data.
4. Load the data into MySQL.
5. Execute SQL queries for analysis.
6. Connect Power BI to MySQL for visualization.

1. Clone the repository.
2. Install the required Python packages.
3. Create a MySQL database.
4. Run the SQL table creation scripts.
5. Execute the ETL Python scripts.
6. Open the Power BI dashboard and connect to the populated database.
